# MotoCare Figma Design References

Use these screenshots as the visual reference for the frontend UI.

- home-page.png: landing/home page style
- booking-page.png: booking service page style
- login-page.png: login page style
- register-page.png: register page style
- service-detail-page.png: service detail style

Main design direction:
- dark automotive website
- black/dark gray background
- orange accent
- premium motorcycle garage feel
- strong bold typography
- dark service cards
- orange buttons
- thin orange borders
- clean spacing

For pages not shown in the screenshots, adapt the same visual language.
Do not remove existing features or break API logic.
